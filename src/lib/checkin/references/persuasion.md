# Persuasão & Gatilhos Mentais

## 7 Princípios de Cialdini no Check-in

| Princípio | Aplicação Concreta | Slide |
|-----------|-------------------|-------|
| **Reciprocidade** | Entregar insights valiosos ANTES de pedir decisões ou aprovação de budget | Insights (antes de Próximos Passos) |
| **Compromisso/Consistência** | Mostrar progresso das ações comprometidas na semana anterior | Contexto Rápido (badge "X de Y concluídas") |
| **Prova Social** | Benchmarks do setor: "Empresas similares alcançam CPL de R$ X" | Benchmark + Análise |
| **Autoridade** | Dados de APIs oficiais, frameworks reconhecidos como badges | Headers com badges de framework |
| **Escassez** | "Janela de oportunidade: público LAL 1% tem ~200K — saturação em X semanas" | Próximos Passos (urgência) |
| **Afinidade (Liking)** | Linguagem personalizada com termos do nicho, design com cores do cliente | Toda apresentação |
| **Unidade** | "Nós, juntos, como parceiros de crescimento" — usar "nós" não "vocês" | Capa e Encerramento |

## Gatilhos Mentais — Frases-Modelo

Usar estas estruturas ao escrever textos dos slides:

| Gatilho | Frase-Modelo |
|---------|-------------|
| **Urgência** | "CPC subiu [X]% — cada dia sem otimização custa R$ [Y] a mais" |
| **Autoridade** | "Segundo dados do Meta Ads Benchmark Q1/2026, o CPL médio do setor é R$ [X]" |
| **Prova Social** | "Empresas do setor de [nicho] têm CPL médio de R$ [X] — você está R$ [Y] abaixo/acima" |
| **Escassez** | "Público lookalike 1% tem apenas ~[N]K pessoas — saturação estimada em [X] semanas" |
| **Reciprocidade** | "Identificamos [N] oportunidades que podem reduzir seu CPL em até [X]%" |
| **Novidade** | "Nova funcionalidade do Meta Ads permite [X] — recomendamos testar esta semana" |
| **Antecipação** | "Na próxima semana, lançaremos o teste A/B que pode ser o game-changer da conta" |
| **Exclusividade** | "Análise personalizada baseada nos dados exclusivos da sua conta" |
| **Contraste** | "Semana passada: R$ [A]/lead. Esta semana: R$ [B]/lead — [contexto]" |
| **Fluência Cognitiva** | Design limpo, hierarquia visual clara, dados digestíveis em 3 segundos |

## Sequência Neuromarketing dos Slides (Pain → Gain → CTA)

A ordem dos slides segue uma estrutura persuasiva intencional:

```
1. IMPACTO 3 SEGUNDOS  → Capa premium (autoridade visual instantânea)
2. DIAGNOSTICAR A DOR   → Gargalo TOC (o que está limitando o crescimento?)
3. MOSTRAR EVIDÊNCIAS   → KPIs com variação (os dados confirmam o diagnóstico)
4. REVELAR OPORTUNIDADES → Insights ("aqui está o que a maioria não vê")
5. PLANO DE AÇÃO        → Próximos Passos (caminho claro para resolver)
6. CRIAR COMPROMISSO    → Encerramento (próximo check-in agendado + tópicos previstos)
```

Esta sequência ativa os circuitos cerebrais de "dor → alívio → ação" que são mais eficazes para gerar comprometimento do que simplesmente mostrar dados.

## Regras de Linguagem

1. **Nunca apenas informar** — sempre contextualizar ("O CPL subiu 5%, MAS ainda está abaixo da meta")
2. **Usar verbos de ação** — "Escalar", "Testar", "Renovar", "Otimizar" (não "seria bom considerar")
3. **Quantificar impacto** — "Economia estimada de R$ X/mês" (não "pode melhorar")
4. **Personalizar ao nicho** — usar termos do setor do cliente
5. **Criar narrativa** — conectar slides como capítulos de uma história (StoryBrand)
